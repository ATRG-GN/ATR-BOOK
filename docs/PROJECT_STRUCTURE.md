# โครงสร้างโฟลเดอร์ของโครงการ ATR-BOOK (สถานะปัจจุบัน)

เอกสารนี้อธิบายโครงสร้างโค้ดที่มีอยู่จริงใน repository ปัจจุบัน
เพื่อลดความคลาดเคลื่อนระหว่างเอกสารกับโค้ดที่รันได้

```text
atr_book/
├── lib/
│   └── main.dart                     # จุดเริ่มแอปและหน้ารายการฟีเจอร์
├── test/
│   └── widget_test.dart              # ชุดทดสอบ Widget สำหรับหน้าหลัก
├── docs/
│   ├── ATR-BOOK_SYSTEM_ARCHITECTURE.md  # วิสัยทัศน์เชิงสถาปัตยกรรมระยะยาว
│   └── PROJECT_STRUCTURE.md          # เอกสารโครงสร้างจริงของโครงการ
├── android/ ios/ linux/ macos/
├── web/ windows/                     # โครงสร้างแพลตฟอร์มมาตรฐานจาก Flutter
├── pubspec.yaml                      # dependency และ metadata โปรเจกต์
└── README.md                         # ภาพรวมโครงการ
```

## หมายเหตุสำคัญ
- โครงสร้างแบบ Feature-First + Clean Architecture ในเอกสารสถาปัตยกรรมเป็น **แผนเป้าหมาย**
  ไม่ใช่โครงสร้างที่ถูกสร้างครบแล้วในโค้ดปัจจุบัน
- หากเริ่มเพิ่มฟีเจอร์ใหม่ แนะนำค่อย ๆ ย้ายจากโครงสร้างปัจจุบันไปสู่โครงสร้างเป้าหมาย
  โดยอัปเดตเอกสารนี้ทุกครั้งที่มีการเพิ่มโฟลเดอร์หลัก
